# Tune Up 波形チャート軸制御改善プラン

**Date:** 2026-02-10
**Status:** IN PROGRESS

## Context

Tune Up モードの波形チャートで、500ms ポーリングごとにチャートオプションが完全再構築されるため、ユーザーのズーム状態がリセットされる。また Y 軸がオートレンジで使いにくい。

**要件:**
- X 軸: waveform 長を最大値として固定。ズーム状態はポーリング間で保持
- Y 軸: デフォルト -20000〜20000。ズーム状態はポーリング間で保持

## 解決策

**対象ファイル:** `web/operator-ui/src/app/pages/waveform/waveform.component.ts`

histogram-chart コンポーネントで実績のある `[options]` + `[merge]` パターンを適用:
- `[options]` — 初期チャート構造（チャンネル切替時にリセット）
- `[merge]` — データ更新（500ms ごと）。`dataZoom` を含めないためズーム状態が保持される

### ズーム動作

| 操作 | X 軸 | Y 軸 |
|------|------|------|
| 初期表示 | 0 〜 waveform length | -20000 〜 20000 |
| Shift+スクロール | X ズーム | 変化なし |
| Ctrl+スクロール | 変化なし | Y ズーム |
| ポーリング更新 | ズーム保持 | ズーム保持 |
| チャンネル切替 | リセット | リセット |
